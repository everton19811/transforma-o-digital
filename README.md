## 🎯 Sobre este repositório

Este repositório reúne **cases práticos de Transformação Digital** que percorrem todo o ciclo de vida do projeto — da **iniciação** ao **encerramento** — aplicando boas práticas de:

- **PMBOK 7** (5 grupos de processo)
- **Governança de TI** (ITIL v4, COBIT)
- **Arquitetura de Soluções** (Cloud, Integrações, Microserviços)
- **Metodologias Ágeis** (Scrum, Kanban, SAFe)
- **Governança de Dados & LGPD**

Cada pasta é **autoexplicativa** e pode ser usada como material de estudo, referência técnica ou template para novos projetos.

## 📁 Estrutura padrão de cada case

Todos os cases seguem a mesma estrutura modular, replicando as fases do PMBOK:

```text
case-<nome>/
├── README.md                      # Visão geral, contexto, resultados
├── 01-iniciacao/                  # Business Case, Charter, Stakeholders
├── 02-planejamento/               # EAP, Cronograma, Riscos, Orçamento, Arquitetura
├── 03-execucao/                   # Planos de execução, integração, migração
├── 04-monitoramento-controle/     # EVM, Status Reports, KPIs, Change Requests
├── 05-encerramento/               # Lições Aprendidas, Post-Mortem, Aceite Final
└── diagramas/                     # Mermaid: Gantt, EAP, Arquitetura, Fluxos
