# Adoção de Assinatura Digital Docusign para Contratos de Fornecedores

> Case do portfólio de Everton Silva — Gerente de Projetos / Analista Funcional Sr. / Analista de Negócios

## Visão Geral
Implantar Docusign eSignature integrado ao SAP Ariba e Jurídico para eliminar assinaturas físicas de contratos com fornecedores, reduzindo o ciclo de contratação em 70% e garantindo validade jurídica (ICP-Brasil/MP 2.200-2).

## Contexto
Processo atual 100% físico: impressão, coleta de assinaturas, malote entre filiais, arquivamento físico. Ciclo médio de 21 dias, alto risco de extravio e falta de rastreabilidade.

## Sistemas / Tecnologias
- Docusign eSignature
- Docusign CLM
- SAP Ariba
- SAP MM
- GED SharePoint
- Active Directory
- ICP-Brasil (certificado A1/A3)

## KPIs Alcançados
- Ciclo de contratação: 21d → 6d
- Custo por contrato: -80% (papel, malote, cartório)
- Contratos com rastreabilidade: 100%
- Adesão de fornecedores: 95% em 6 meses
- Redução de contratos extraviados: 100%

## Principais Riscos
- Resistência de fornecedores de pequeno porte
- Aderência jurídica em contratos internacionais
- Integração SAP Ariba com Docusign Connect
- Adequação LGPD para dados de signatários
